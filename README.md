# FaceScore

FaceScore is a Tauri v2 desktop MVP for uploading a face photo, sending it to Claude Vision with a user-provided API key, rendering a structured appearance report, and exporting the report to PDF.

## Stack

- Tauri v2
- React
- TypeScript
- Vite

## Local Development

```powershell
npm install
npm run dev
```

Run the desktop shell after installing the Tauri prerequisites, including Rust:

```powershell
npm run tauri dev
```

Build the frontend:

```powershell
npm run build
```
